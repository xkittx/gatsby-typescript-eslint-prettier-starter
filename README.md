# Gatsby TypeScript ESLint Prettier Starter

## 📁 Use

- yarn
- Gatsby
- TypeScript
- ESLint
- Prettier

## ⌨ Command

### ESLint

ESLint syntax check execution

```sh
yarn lint
```

Applying ESLint syntax

```sh
yarn lint:fix
```

### Prettier

Apply code formatting

```sh
yarn format
```

## ♦ About this repository

This repository is the starter theme I use when using gatsby.
